## 后端小服务
### 定时任务管理服务
#### 描述
 接受定时任务, 到时间后按照设定参数进行请求
 可接受相对时间(...s之后or...h之后 (年月日时分秒等)一类)以及
 绝对时间(xxxx-xx-xx) or (xxxx-xx-xx xx:xx:xx)以及
 绝对时间循环
 相对时间循环
 相对转绝对时间循环
#### 作用
 可以作为小中枢, 程序内可以用Golang自身特性解决, 另外暴露http接口
#### 设计
 接收事件首先测量时间差, 时间长则丢到队列, 短则开线程处理

 如果是循环, 线程中首先处理
